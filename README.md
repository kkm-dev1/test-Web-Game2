# 🐛 지렁이 키우기 게임

Flask를 사용한 재미있는 지렁이 키우기 웹 게임입니다. 구글 검색에 노출되도록 SEO 최적화가 되어 있습니다.

## 🎮 게임 소개

귀여운 지렁이를 키우는 재미있는 무료 온라인 게임입니다. 지렁이를 클릭하거나 버튼을 눌러서 경험치를 얻고 레벨업하세요! 게임 진행 상황은 자동으로 저장되어 다음에 접속해도 계속 이어서 플레이할 수 있습니다.

## ✨ 주요 기능

- **클릭하여 키우기**: 지렁이를 클릭하거나 버튼을 눌러 경험치 획득
- **레벨업 시스템**: 경험치를 모아 레벨업하고 지렁이가 성장
- **자동 저장**: 로컬 스토리지를 이용한 영구 저장 기능
- **자동 먹이주기**: 자동으로 지렁이를 키우는 기능
- **반응형 디자인**: 모바일과 데스크톱 모두 지원
- **SEO 최적화**: 구글 검색 엔진에 노출되도록 최적화

## 📦 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 🚀 실행 방법

1. 터미널에서 다음 명령어 실행:
```bash
python app.py
```

2. 웹 브라우저에서 다음 주소로 접속:
```
http://localhost:5000
```

## 🎯 게임 방법

1. **지렁이 클릭하기**: 지렁이를 클릭하면 경험치를 얻습니다
2. **먹이주기 버튼**: "먹이주기" 버튼을 눌러도 경험치를 얻을 수 있습니다
3. **레벨업**: 경험치가 충분하면 자동으로 레벨업합니다
4. **자동 먹이주기**: "자동 먹이주기" 버튼을 눌러 자동으로 키울 수 있습니다
5. **게임 초기화**: "게임 초기화" 버튼으로 처음부터 다시 시작할 수 있습니다

## 📊 게임 시스템

- **경험치 시스템**: 레벨이 올라갈수록 더 많은 경험치가 필요합니다
- **성장 시스템**: 레벨업할 때마다 지렁이가 커집니다
- **통계 추적**: 클릭 횟수, 총 경험치 등 통계를 확인할 수 있습니다

## 🔍 SEO 최적화

이 게임은 구글 검색에 노출되도록 다음과 같은 SEO 최적화가 적용되어 있습니다:

- **메타 태그**: 검색 엔진을 위한 메타 설명 및 키워드
- **구조화된 데이터**: JSON-LD 형식의 구조화된 데이터
- **Open Graph 태그**: 소셜 미디어 공유 최적화
- **시맨틱 HTML**: 검색 엔진이 이해하기 쉬운 HTML 구조
- **반응형 디자인**: 모바일 친화적 디자인

## 📁 파일 구조

- `app.py`: Flask 서버 (간단한 라우팅)
- `templates/game.html`: 게임 웹 페이지 (HTML, CSS, JavaScript 포함)
- `requirements.txt`: 필요한 Python 패키지 목록
- `README.md`: 프로젝트 설명서

## 🌐 배포 방법

구글 검색에 노출되려면 실제 웹 서버에 배포해야 합니다:

1. **호스팅 서비스 선택**: Heroku, AWS, Google Cloud Platform 등
2. **도메인 연결**: 고유한 도메인 이름 설정
3. **SSL 인증서**: HTTPS 프로토콜 사용 (검색 엔진 최적화에 중요)
4. **Google Search Console 등록**: 구글에 사이트 등록
5. **sitemap.xml 생성**: 검색 엔진 크롤링을 위한 사이트맵 생성
